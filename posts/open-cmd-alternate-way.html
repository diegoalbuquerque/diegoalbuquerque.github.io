:: TL; DR;

Acesse um prompt de comando no windows usando diretamente uma caixa de dialogo que permita selecionar arquivos:
- Ache a caixa de diálogo através de propriedade de atalhos, arquivos, adaptadores de rede, etc.
- Navegue até a pasta c:\windows\system32
- Encontre o arquivo cmd.exe e arraste para cima dele qualquer outro arquivo visível na caixa de diálogo

Algumas vezes precisamos efetuar algumas ações no windows que em um primeiro momento está bloqueada por determinadas configurações, como por exemplo, copiar um arquivo de um pendrive ou mesmo executar algum comando do sistema, etc.
Em um ambiente assim, se conseguirmos executar um cmd.exe podemos abrir várias possibilidades, como até mesmo copiar o arquivo do pendrive montado mas o qual você não teria acesso de forma direta através de um Explorer, etc.
A ideia é executar um cmd.exe de uma forma inesperada e para isso usaremos uma funcionalidade presente no windows, inclusive no windows 10 atual, que permite abrirmos um cmd diretamente de uma caixa de diálogo de arquivos.

:: Primeiro Passo

O primeiro passo então é achar, no sistema, de alguma forma, algum meio de ter acesso a uma caixa de seleção de arquivos.
As possibilidades podem ser muitas, sua tarefa é achar algum lugar onde abra uma caixa de selecionar arquivos. Pode ser de drivers, de configuração, qualquer coisa.
Alguns modos como forma de exemplo.
*Exemplo 1 *
Através das propriedades de um arquivo. Pode-se criar, se for possível, um novo documento ou pasta na área de trabalho ou usar um já existente. Até mesmo um atalho serve.

Uma vez aberta a caixa de diálogo, navegue para a pasta c:\windows\system32 e localize o cmd.exe.

*Exemplo 2*
Através de um atalho.

*Exemplo 3*
Se você não tiver nenhum acesso a qualquer arquivo ou mesmo não pode acessar as propriedades, mas tem acesso ao ícone de identificação de rede.


:: Segundo passo

Com a caixa de diálogo aberta, certifique-se de navegar até a pasta c:\windows\system32 e verificar se está listando todos os arquivos.
_Para listar todos os arquivos, geralmente aparece uma caixa de seleção onde se pode escolher, ou pode-se digitar *.* diretamente no campo nome._
Localize o arquivo cmd.exe
Uma vez localizado o cmd.exe , agora é arrastar qualquer arquivo presente na caixa de diálogo, por exemplo, neste caso usamos o comp.exe, para cima do cmd.exe e ver o prompt de comando abrir.

Executar o que precisa. Fim. ;D
